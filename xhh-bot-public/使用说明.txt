一、这是什么？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

一个开源的小黑盒自动回复机器人。
当别人在小黑盒 @ 你的账号时，机器人会自动：
  1. 读取帖子内容和图片
  2. 调用 AI 生成回复
  3. 自动回复评论

任何人都可以自己部署。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
二、准备工作
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

你需要准备两样东西：

【1】Node.js（JavaScript 运行环境）
  - 下载：https://nodejs.org/ 下载 LTS 版
  - 安装：一路下一步就行
  - 验证：打开 cmd 输入 node --version，有版本号就行

【2】一个 AI 接口的 API Key（三选一）

  选项A：阿里云通义千问（推荐，国内直连）
    注册：https://阿里云.com
    找到：模型服务灵积 -> API Key 管理
    创建 API Key

  选项B：DeepSeek
    注册：https://platform.deepseek.com
    充值几块钱就够了

  选项C：其他 OpenAI 兼容接口
    随便什么，只要能用 /v1/chat/completions 就行


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
三、获取小黑盒 Cookie
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开 Edge 或 Chrome 浏览器
2. 访问 https://www.xiaoheihe.cn 并登录你的账号
3. 按键盘上的 F12 键（打开开发者工具）
4. 点击顶部 "Network"（网络）标签
5. 在过滤输入框里输入：api.xiaoheihe
6. 按 F5 刷新页面
7. 点击出现的第一个请求
8. 在右侧找到 "Request Headers"（请求标头）
9. 找到 "Cookie" 这一行
10. 从 Cookie 里提取三个值：

   user_pkey        → 一串乱码一样的字符
   user_heybox_id   → 一串数字（你的小黑盒 UID）
   x_xhh_tokenid    → 另一串乱码

   把它们记下来，下一步要用。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
四、配置
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 把 xhh-config.example.json 重命名为 xhh-config.json
2. 用记事本打开 xhh-config.json
3. 填写配置：

  (1) cookie 部分（填上一步从浏览器复制的三个值）：
    "user_pkey": "你复制的 user_pkey",
    "heybox_id": "你复制的 heybox_id",
    "x_xhh_tokenid": "你复制的 x_xhh_tokenid",

  (2) ai 部分：
    "model": "qwen-turbo",     ← AI 模型名
    "baseUrl": "https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions",
                                   ← API 地址
    "token": "sk-xxxxxxxx",    ← 你的 API Key

    "prompt" 是 AI 的角色设定，可以改成你想要的语气

  (3) database 部分（默认用 SQLite，不用改）：
    "type": "sqlite"           ← sqlite 或 postgresql


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
五、启动机器人
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开 cmd 或 PowerShell
2. 进入这个文件夹：

   cd D:\你的路径\xhh-bot

3. 安装依赖（只需要第一次）：

   npm install

4. 启动：

   node src/index.js

或者直接双击 start.bat 也行。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
六、查看统计
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在 cmd 里运行：

   node src/index.js --stats

会显示：
  - 最近 7 天每天多少人 @
  - 今天每个小时多少人 @
  - 所有 @ 过你的用户列表

七、常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q：启动后显示 "验证参数错误"？
A：Cookie 过期了。重新从浏览器 F12 复制 Cookie 更新配置。

Q：显示 "hkey 不能为空"？
A：同样 Cookie 过期了，重新获取。

Q：显示 "show_captcha"？
A：需要完整的 Cookie（包括 user_pkey 和 heybox_id），
   不只是 x_xhh_tokenid。

Q：AI 返回空或报错？
A：检查 API Key 是否有效，余额是否充足。

Q：想用 PostgreSQL？
A：把 database.type 改成 "postgresql"，填好数据库地址、
   用户名和密码。需要先 CREATE TABLE。
