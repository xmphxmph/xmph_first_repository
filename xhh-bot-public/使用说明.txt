一、这是什么？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

一个开源的小黑盒自动回复机器人。
当别人在小黑盒 @ 你的账号时，机器人会自动：
  1. 读取帖子内容和图片
  2. 调用 AI 生成回复
  3. 自动回复评论

任何人都可以自己部署。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
二、准备工作
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

你需要准备两样东西：

【1】Node.js（JavaScript 运行环境）
  - 下载：https://nodejs.org/ 下载 LTS 版
  - 安装：一路下一步就行
  - 验证：打开 cmd 输入 node --version，有版本号就行

【2】一个 AI 接口的 API Key（三选一）

  选项A：阿里云通义千问（推荐，国内直连）
    注册：https://阿里云.com
    找到：模型服务灵积 -> API Key 管理
    创建 API Key

  选项B：DeepSeek
    注册：https://platform.deepseek.com
    充值几块钱就够了

  选项C：其他 OpenAI 兼容接口
    随便什么，只要能用 /v1/chat/completions 就行


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
三、从浏览器里拿 Cookie（让机器人登录你的账号）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

机器人要用你的小黑盒账号来回复，所以需要从浏览器里复制
三样东西（就是 Cookie 里的三个值）。跟着下面一步步来：
1.打开小黑盒网页，登录你想要当成机器人的帐号
2.按F12
3.选择"网络"
4.在筛选器栏输入 api.xiaoheihe.cn 进行过滤
5.筛选后点F5刷新网页
6.你需要找一个请求，在标头中向下翻，找到cookie
7.Cookie 里有这三样东西才算全（如果不全，找到其中一个先记下来，然后找另外的请求）：
  user_pkey= xxxx...; 
  user_heybox_id= 一串数字; 这玩意就是你的机器人账号的uid，自己在主页直接看也行
  x_xhh_tokenid= xxx...
将这三个数字保存下来，等会用
>>如果现在一切正常，那你就可以直接前往第四点<<

tips：想自己动手改文件也行，操作如下：

1. 把 xhh-config.example.json 改名成 xhh-config.json
2. 用记事本打开它，里面大概长这样：

   ┌──────────────────────────────────────────┐
   │ {                                        │
   │   "cookie": {                            │
   │     "user_pkey": "双引号里填这串",       │
   │     "heybox_id": "双引号里填这个数字",   │
   │     "x_xhh_tokenid": "双引号里填这串"    │
   │   },                                     │
   │   "ai": {                                │
   │     "token": "双引号里填你的AI密钥"      │
   │   }                                      │
   │ }                                        │
   └──────────────────────────────────────────┘

3. 看到有双引号的地方，把引号里面的内容换成你自己的
   （注意别删掉逗号和花括号，改坏文件会跑不起来）


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
四、启动机器人
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

搞完配置就能跑了：

  双击 setup.bat     ← 第一次用，跟着提示填 Cookie 和 AI 密钥
  然后双击 start.bat ← 以后直接双击这个启动

或者在命令行里也行：

  npm install        ← 装依赖，第一次运行前做一次就行
  node src/setup.js  ← 初始化向导（填 Cookie 和 AI 密钥）
  node src/index.js  ← 启动机器人

启动后让它挂在那里就行，它会每隔 30 秒检查一次有没有人 @ 你。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
五、看看有谁 @ 过我（统计）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

想看看机器人多受欢迎？在命令行里运行：

  node src/index.js --stats

会显示：
  - 最近 7 天，每天有多少人 @ 了你的机器人
  - 今天每个小时有多少人 @
  - 所有 @ 过你的人，每人 @ 了几次

六、常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q：启动后提示"验证参数错误"？
A：Cookie 过期了。重新按第三步从浏览器 F12 里复制新的 Cookie，
   然后重新运行 setup.bat 更新配置。

Q：提示"hkey 不能为空"？
A：一样的问题，Cookie 过期了。重新弄一遍。

Q：提示"show_captcha"？
A：说明 Cookie 没拿全，你把 user_pkey 和 heybox_id 也填上，
   光填 x_xhh_tokenid 不够。

Q：AI 不回消息或者报错？
A：检查你填的 AI 密钥有没有过期、钱包里有没有钱。

Q：想用自己的数据库（PostgreSQL）？
A：运行 setup.bat，选 PostgreSQL，填上数据库地址、用户名密码就行。

Q:想要更改角色设定？
A:1.重新运行setup.bat;
  2.用记事本打开 xhh-config.json，找到你想改的部分直接改：

改 Cookie → 找到 "cookie" 那一段，替换成从浏览器 F12 新复制的值
改角色设定 → 找到 "prompt"，把双引号里面的文字换成新的
换 AI 模型 → 找到 "model" 和 "token"，换成新的
换数据库 → 找到 "database"，把 type 改成 "postgresql" 或 "sqlite"
改完保存，重启机器人（关掉旧的命令行窗口，重新双击 start.bat）就行。

七、各文件是干什么的
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

xhh-config.json          ← 你的配置（Cookie + AI 密钥），自己改
xhh-config.example.json  ← 配置模板，带注释说明
setup.bat                ← 初始化向导，双击运行填配置
start.bat                ← 启动机器人，双击运行
使用说明.txt             ← 你现在看的这个文件
README.md                ← 英文版说明

src/ 里面的程序文件（不懂编程不用动）：
├── setup.js             ← 初始化向导程序
├── index.js             ← 主程序
├── api.js               ← 小黑盒接口
├── ai.js                ← AI 调用
└── db.js                ← 数据库
